# 首页仪表盘

## 概述

展示库存系统的核心数据概览和快捷入口。

## 输入输出

| 类型 | 说明 |
|------|------|
| 输入 | 用户登录状态 |
| 输出 | 统计卡片、快捷操作入口 |

## 核心逻辑

1. 加载用户信息和权限
2. 根据权限显示对应功能入口
3. 展示库存概览数据

## 关键代码

```tsx
// frontend/src/pages/Dashboard.tsx
const Dashboard = () => {
  const { user } = useUser()
  // 根据权限渲染功能卡片
}
```

## 依赖关系

- UserContext 用户上下文
- stock API 获取统计数据

## 注意事项

- 未登录用户自动跳转登录页
