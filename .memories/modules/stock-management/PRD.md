# 产品需求文档 - 库存管理

## 业务目标

实现库存的全生命周期管理，包括入库、出库、查询、预警和盘点。

## 目标用户

| 角色 | 权限 |
|------|------|
| 入库管理员 | 入库操作 |
| 出库管理员 | 出库操作 |
| 仓库管理员 | 全部库存操作 |
| 财务 | 只读查询 |

## 核心数据模型

| 模型 | 说明 |
|------|------|
| Stock | 库存表 |
| StockIn | 入库记录 |
| StockOut | 出库记录 |
| StockWarning | 预警记录 |
| StockCountTask | 盘点任务 |
