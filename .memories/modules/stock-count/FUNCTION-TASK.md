# 盘点任务管理

## 概述

管理盘点任务的完整生命周期。

## 输入输出

| 类型 | 说明 |
|------|------|
| 输入 | 盘点范围、实盘数量 |
| 输出 | 盘点结果、盘盈盘亏记录 |

## 核心逻辑

1. 创建任务，生成盘点明细
2. 逐项录入实盘数量
3. 完成时计算差异，生成调整记录

## 依赖关系

## 注意事项

- 盘亏自动生成出库记录
