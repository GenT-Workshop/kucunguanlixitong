# 登录功能

## 概述

用户通过用户名和密码登录系统，验证成功后返回用户信息和权限列表。

## 输入输出

| 类型 | 说明 |
|------|------|
| 输入 | `username`, `password` |
| 输出 | 用户信息、角色列表、权限列表 |

## API 接口

- **路径**: `POST /api/login/`
- **请求体**: `{ "username": "xxx", "password": "xxx" }`

## 核心逻辑

1. 验证用户名密码
2. 查询用户关联的角色（UserRole）
3. 查询角色关联的权限（RolePermission）
4. 返回用户信息和权限列表
